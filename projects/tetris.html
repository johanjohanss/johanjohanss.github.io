Tetris Game Study

A very simple tetris game made to practice my javascript skills. I also made my own (kind of experimental) version of the tetris song to go along with this. Start the game and use the arrow keys to rotate and move the shapes down. Your highscore is saved in a cookie so that you can try to beat it later (although you may have to refresh the page to see it when you come from the start page). This project is not responsive and there are no mobile controls, so it will not work on phones or tablets.
